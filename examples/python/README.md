# Python examples

```bash
pip install ghostcrawl
export GHOSTCRAWL_API_KEY="YOUR_API_KEY"
```

| File | What it shows |
|------|---------------|
| `scrape.py` | Scrape one page (sync). |
| `crawl.py` | Start a crawl run and read its status. |
| `extract.py` | Extract structured JSON against a schema. |
| `async_scrape.py` | Scrape concurrently with the async client. |

Run any example:

```bash
python scrape.py
```
